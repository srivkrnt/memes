# Format details:

For meme generation, we currently have the following formats. It gives the textual representation of the memes after their creation.

### Format 1:

```
Type 1: __________________
        |  Text on top   |
        |                |
        |                |
        |                |
        |________________|


Type 2: __________________
        |                |
        |                |
        |                |
        | Text in bottom |
        |________________|


Type 3: __________________
        |  Text on top   |
        |                |
        |                |
        | Text in bottom |
        |________________|
```

### Format 2:

```	   
        __________________
        |  Text on top   |
        |                |
        |                |
        |________________|
        |                |
        |                |
        | Text in bottom |
        |________________|
```

```

        |    Text on the top     |
        |           |            |

Type 4: __________________________
```
